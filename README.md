##

1. **Clone the Repository**

   ```sh
   git clone https://github.com/TeaByte/sqlite-viewer.git
   cd sqlite-viewer
   ```

2. **Install Dependencies**

   ```sh
   npm install
   cargo install tauri-cli
   ```

3. **Start Dev Server**

   ```sh
   cargo tauri dev
   ```
